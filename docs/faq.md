# Часто задаваемые вопросы (FAQ)

## Общие вопросы

### Нужны ли API-ключи или платные сервисы?

Нет. Claude Marketing Suite работает полностью через Claude Code. Вам нужна только подписка Claude Pro, Team или Enterprise.

### На каких языках работают скиллы?

Скиллы работают на русском и английском. Язык вывода определяется автоматически по языку сайта или может быть указан явно в запросе.

### Можно ли использовать скиллы для клиентских проектов?

Да. Лицензия MIT позволяет коммерческое использование. Выходные файлы принадлежат вам.

## Установка

### Какая версия Claude Code нужна?

Рекомендуется последняя версия. Установите или обновите: `npm install -g @anthropic-ai/claude-code`.

### Скиллы не распознаются после установки

1. Убедитесь, что запустили `claude install-skills` из корня репозитория
2. Перезапустите Claude Code
3. Проверьте, что `.skill` файлы на месте в папке `skills/`

## Использование

### Обязательно ли запускать скиллы в определённом порядке?

Нет. Каждый скилл работает автономно. Но если запускать в рекомендуемом порядке (audit -> brand -> audience -> контент), качество вывода будет выше за счёт кросс-скилл интеграции.

### Можно ли запустить все скиллы одной командой?

Да: `/aim report https://your-site.com`. Мета-оркестратор запустит все скиллы последовательно.

### Скилл создал файл, но он неполный

Попробуйте:
1. Сначала запустить базовые скиллы (`/aim audit`, `/aim brand`, `/aim audience`)
2. Затем повторить нужный скилл -- он подхватит данные из базовых файлов

### Можно ли анализировать сайт конкурента?

Да. Передайте URL конкурента в любой скилл: `/aim audit https://competitor.com`.

### Перезаписываются ли файлы при повторном запуске?

Да. Каждый запуск создаёт файл заново. Если хотите сохранить предыдущую версию -- переименуйте файл перед повторным запуском.

## Технические вопросы

### Куда сохраняются выходные файлы?

В текущую рабочую директорию Claude Code (обычно корень проекта).

### Отправляются ли данные на сторонние серверы?

Нет. Данные обрабатываются через стандартный API Claude (Anthropic). Никакие сторонние сервисы не используются.

### Можно ли кастомизировать скиллы?

Да. Отредактируйте `SKILL.md` в папке нужного скилла. Это промпт-инструкция, которую Claude получает при запуске.

### Работает ли с Claude Desktop?

Скиллы разработаны для Claude Code (CLI). В Claude Desktop они не поддерживаются.
