# Пример: SaaS-компания

## Сценарий

Вы запускаете B2B SaaS-продукт для управления проектами. Сайт: `https://taskflow.io`. Нужно провести полный маркетинговый анализ и подготовить контент-стратегию.

---

## Шаг 1: Маркетинговый аудит

```
/aim audit https://taskflow.io
```

Получаем `MARKETING-AUDIT.md` со скорингом по 12 категориям. Выясняем, что слабые места -- позиционирование (52/100) и CTA (45/100).

## Шаг 2: CRO-аудит лендинга

```
/aim landing https://taskflow.io
```

`LANDING-CRO.md` показывает: hero-секция не доносит ценность, social proof отсутствует, форма регистрации слишком длинная.

## Шаг 3: Конкурентный анализ

```
/aim competitors https://taskflow.io
```

`COMPETITOR-REPORT.md` выявляет 4 конкурентов, находит незанятую нишу: интеграция с мессенджерами для небольших команд.

## Шаг 4: Голос бренда

```
/aim brand https://taskflow.io
```

`BRAND-VOICE.md` определяет архетип «Помощник», формирует словарь бренда и тональность для всех каналов.

## Шаг 5: Исследование аудитории

```
/aim audience https://taskflow.io
```

`AUDIENCE-RESEARCH.md` выделяет 3 макросегмента: тимлиды в стартапах, PM в среднем бизнесе, фрилансеры-консультанты.

## Шаг 6: Email-последовательности

```
/aim emails
```

`EMAIL-SEQUENCES.md` генерирует 4 цепочки: welcome (5 писем), onboarding (7 писем), upgrade (4 письма), win-back (3 письма).

## Шаг 7: Контент-план

```
/aim social
```

`SOCIAL-CALENDAR.md` создаёт 30-дневный план для LinkedIn и Twitter с акцентом на продуктивность и управление командой.

## Шаг 8: Рекламные кампании

```
/aim ads
```

`AD-CAMPAIGNS.md` генерирует кампании для Google Search, LinkedIn Ads и Meta с таргетингом на каждый сегмент.

## Шаг 9: Заголовки

```
/aim headlines
```

`HEADLINES.md` -- 100 заголовков для лендинга, рекламы и email.

## Результат

9 готовых документов с полной маркетинговой стратегией, конкретными текстами и планом действий. Время: ~30 минут вместо 2-3 недель работы маркетолога.
