# Пример: Интернет-магазин (E-commerce)

## Сценарий

Интернет-магазин органической косметики `https://naturaskin.ru`. Цель: увеличить конверсию сайта, выстроить email-маркетинг и запустить рекламу.

---

## Шаг 1: Аудит сайта

```
/aim audit https://naturaskin.ru
```

`MARKETING-AUDIT.md` выявляет: слабая карточка товара, нет отзывов на главной, отсутствует блог.

## Шаг 2: CRO-аудит главной страницы

```
/aim landing https://naturaskin.ru
```

`LANDING-CRO.md` даёт конкретные рекомендации: добавить UGC-фото, вынести бестселлеры выше, упростить навигацию.

## Шаг 3: Анализ конкурентов

```
/aim competitors https://naturaskin.ru
```

`COMPETITOR-REPORT.md` сравнивает с 5 конкурентами, находит возможность: программа рефералов + подписка на регулярные поставки.

## Шаг 4: Голос бренда

```
/aim brand https://naturaskin.ru
```

`BRAND-VOICE.md` формирует стиль: тёплый, экспертный, с акцентом на натуральность и заботу.

## Шаг 5: Целевая аудитория

```
/aim audience https://naturaskin.ru
```

`AUDIENCE-RESEARCH.md` выделяет сегменты: осознанные мамы 28-38, eco-энтузиасты 22-30, женщины 35+ с чувствительной кожей.

## Шаг 6: Email-маркетинг

```
/aim emails
```

`EMAIL-SEQUENCES.md` генерирует: welcome-серию с промокодом, брошенную корзину (3 письма), post-purchase (уход за кожей), реактивацию.

## Шаг 7: Контент для соцсетей

```
/aim social
```

`SOCIAL-CALENDAR.md` создаёт 30-дневный план для Instagram и Telegram: обзоры составов, до/после, рутины ухода, UGC.

## Шаг 8: Рекламные кампании

```
/aim ads
```

`AD-CAMPAIGNS.md` генерирует кампании: Google Shopping, Instagram Stories, ретаргетинг в Meta.

## Шаг 9: Заголовки для акций

```
/aim headlines
```

`HEADLINES.md` -- 100 заголовков для карточек товаров, баннеров и email-рассылок.

## Результат

Полная маркетинговая стратегия для интернет-магазина: от оптимизации сайта до готовых рекламных текстов и email-цепочек.
