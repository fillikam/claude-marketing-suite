# :dart: Landing CRO -- Аудит конверсии лендинга

Проводит посекционный CRO-аудит лендинга, выявляет барьеры конверсии и даёт конкретные рекомендации с оценкой revenue impact.

## Команда

```
/aim landing https://example.com
```

## Выходной файл

`LANDING-CRO.md`

### Ключевые разделы

- **Посекционный разбор** -- hero, benefits, social proof, pricing, CTA, footer
- **Конверсионные барьеры** -- что мешает пользователю совершить действие
- **Рекомендации по каждой секции** -- конкретные изменения текста, дизайна, структуры
- **A/B-тесты** -- гипотезы для тестирования с ожидаемым impact
- **Revenue impact** -- оценка влияния каждого изменения на выручку

## Когда использовать

- Лендинг не конвертирует или конвертирует ниже ожиданий
- Запуск нового продукта или оффера
- Подготовка к рекламной кампании (нужно убедиться, что лендинг готов)

## Входные данные

| Параметр | Обязательный | Описание |
|----------|-------------|----------|
| URL лендинга | Да | Адрес анализируемой страницы |

## Зависимости

Независимый скилл. Не требует других файлов. Его данные используются в сводном отчёте (`/aim report`).

## Установка

Скилл устанавливается вместе со всем набором: `claude install-skills` из корня репозитория.
