# :zap: Headline Generator -- 100 заголовков прямого отклика

Генерирует 100 мощных рекламных заголовков на основе фреймворков Юджина Шварца, матрицы 4U и когнитивных триггеров.

## Команда

```
/aim headlines https://example.com
```

## Выходной файл

`HEADLINES.md`

### Ключевые разделы

- **Заголовки по уровням осведомлённости** -- Most Aware, Product Aware, Solution Aware, Problem Aware, Unaware
- **Группировка по типам** -- выгода, любопытство, страх потери, социальное доказательство, срочность
- **A/B-варианты** -- пары заголовков для тестирования
- **Заголовки по каналам** -- для лендинга, email, рекламы, соцсетей
- **Power words** -- использованные слова-триггеры с пояснениями

## Когда использовать

- Запуск лендинга или рекламной кампании
- Нужно много вариантов для A/B-тестирования
- Рассылка или пост не открывают -- проблема в заголовке
- Запуск нового продукта или курса

## Входные данные

| Параметр | Обязательный | Описание |
|----------|-------------|----------|
| URL сайта | Нет | Если не указан, использует данные из существующих файлов |

## Зависимости

Опционально читает: `BRAND-VOICE.md`, `AUDIENCE-RESEARCH.md`. С этими файлами заголовки точнее попадают в тон бренда и боли аудитории.

## Установка

Скилл устанавливается вместе со всем набором: `claude install-skills` из корня репозитория.
