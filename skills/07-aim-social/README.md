# :calendar: Social Calendar -- 30-дневный контент-план

Создаёт полный 30-дневный контент-план для соцсетей с рубриками, форматами, текстами и хэштегами.

## Команда

```
/aim social https://example.com
```

## Выходной файл

`SOCIAL-CALENDAR.md`

### Ключевые разделы

- **Контент-стратегия** -- рубрики, соотношение типов контента, tone of voice
- **Календарь на 30 дней** -- дата, рубрика, тема, формат, текст поста
- **Хэштеги** -- наборы хэштегов по рубрикам
- **Время публикации** -- рекомендуемое время для каждой платформы
- **Визуальные референсы** -- описание визуалов для каждого поста

## Когда использовать

- Запуск соцсетей с нуля
- Закончились идеи для контента
- Нужен системный подход вместо хаотичных публикаций
- Онбординг SMM-менеджера

## Входные данные

| Параметр | Обязательный | Описание |
|----------|-------------|----------|
| URL сайта | Нет | Если не указан, использует данные из существующих файлов |

## Зависимости

Опционально читает: `BRAND-VOICE.md`, `AUDIENCE-RESEARCH.md`, `MARKETING-AUDIT.md`. С этими файлами контент точнее попадает в тон бренда и интересы аудитории.

## Установка

Скилл устанавливается вместе со всем набором: `claude install-skills` из корня репозитория.
