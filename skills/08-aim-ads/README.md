# :loudspeaker: Ad Campaigns -- Рекламные кампании

Генерирует полные рекламные кампании для Google, Meta, LinkedIn и TikTok с таргетингом, креативами и бюджетами.

## Команда

```
/aim ads https://example.com
```

## Выходной файл

`AD-CAMPAIGNS.md`

### Ключевые разделы

- **Стратегия кампаний** -- цели, воронка, распределение бюджета
- **Google Ads** -- Search, Display, YouTube -- ключевые слова, объявления, расширения
- **Meta Ads** -- Facebook, Instagram -- аудитории, креативы, плейсменты
- **LinkedIn Ads** -- таргетинг по должностям, форматы, тексты
- **TikTok Ads** -- концепции роликов, хуки, CTA
- **Ретаргетинг** -- стратегия возврата посетителей
- **Бюджеты и KPI** -- рекомендуемые бюджеты и целевые метрики

## Когда использовать

- Запуск платной рекламы
- Текущие кампании не окупаются
- Выход на новый канал (LinkedIn, TikTok)
- Нужны готовые тексты объявлений

## Входные данные

| Параметр | Обязательный | Описание |
|----------|-------------|----------|
| URL сайта | Нет | Если не указан, использует данные из существующих файлов |

## Зависимости

Опционально читает: `AUDIENCE-RESEARCH.md`, `BRAND-VOICE.md`, `COMPETITOR-REPORT.md`. С этими файлами таргетинг и креативы значительно точнее.

## Установка

Скилл устанавливается вместе со всем набором: `claude install-skills` из корня репозитория.
