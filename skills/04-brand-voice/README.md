# :performing_arts: Brand Voice -- Анализ голоса бренда

Анализирует голос, тон и сообщения бренда по всем каналам. Создаёт гайдлайны для единообразной коммуникации.

## Команда

```
/aim brand https://example.com
```

## Выходной файл

`BRAND-VOICE.md`

### Ключевые разделы

- **Архетип бренда** -- определение ключевого архетипа и его проявлений
- **Тональность** -- шкала формальности, эмоциональности, экспертности
- **Словарь бренда** -- слова-маркеры, слова-табу, фирменные выражения
- **Do / Don't** -- примеры правильного и неправильного тона
- **Примеры копирайтинга** -- как звучит бренд в разных каналах (сайт, email, соцсети, реклама)

## Когда использовать

- Формирование brand guidelines
- Онбординг новых копирайтеров
- Перед созданием контента (emails, social, ads)
- Ребрендинг

## Входные данные

| Параметр | Обязательный | Описание |
|----------|-------------|----------|
| URL сайта | Да | Адрес сайта для анализа тона коммуникации |

## Зависимости

Опционально читает: `MARKETING-AUDIT.md`, `COMPETITOR-REPORT.md`. Его выходной файл используется скиллами: emails, social, ads, headlines.

## Установка

Скилл устанавливается вместе со всем набором: `claude install-skills` из корня репозитория.
