# :mag: Market Audit -- Полный маркетинговый аудит

Проводит комплексный маркетинговый аудит сайта с скорингом по 12 категориям и приоритезированными рекомендациями.

## Команда

```
/aim audit https://example.com
```

## Выходной файл

`MARKETING-AUDIT.md`

### Ключевые разделы

- **Скоринг по 12 категориям** -- позиционирование, УТП, воронка, CTA, контент, SEO, social proof, дизайн, мобильная версия, скорость, доверие, аналитика
- **Критические проблемы** -- что теряет деньги прямо сейчас
- **Рекомендации с revenue impact** -- каждая рекомендация с оценкой влияния на выручку
- **План действий 30/60/90 дней** -- приоритезированный roadmap
- **Quick wins** -- что можно исправить за 1-2 дня

## Когда использовать

- Первый анализ нового сайта или проекта
- Поиск причин низкой конверсии
- Подготовка к редизайну или ребрендингу

## Входные данные

| Параметр | Обязательный | Описание |
|----------|-------------|----------|
| URL сайта | Да | Адрес анализируемого сайта |

## Зависимости

Независимый скилл. Не требует других файлов. Его выходной файл используется скиллами: brand, audience, emails, social, ads.

## Установка

Скилл устанавливается вместе со всем набором: `claude install-skills` из корня репозитория.
