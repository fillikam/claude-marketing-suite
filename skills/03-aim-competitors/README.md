# :crossed_swords: Competitors -- Конкурентная разведка

Анализирует 3-5 ключевых конкурентов, строит карту рынка, выявляет стратегические окна и точки дифференциации.

## Команда

```
/aim competitors https://example.com
```

## Выходной файл

`COMPETITOR-REPORT.md`

### Ключевые разделы

- **Карта конкурентов** -- кто, что предлагает, ценообразование, позиционирование
- **Сравнительная таблица** -- фичи, цены, сильные/слабые стороны
- **SWOT по каждому конкуренту** -- структурированный анализ
- **Стратегические окна** -- незанятые ниши и возможности
- **Рекомендации по дифференциации** -- как выделиться на фоне конкурентов

## Когда использовать

- Выход на новый рынок
- Пересмотр позиционирования
- Подготовка к инвестиционному раунду
- Конкуренты начали отбирать долю рынка

## Входные данные

| Параметр | Обязательный | Описание |
|----------|-------------|----------|
| URL сайта | Да | Адрес вашего сайта (конкуренты определяются автоматически) |

## Зависимости

Независимый скилл. Его выходной файл используется скиллами: brand, audience, ads.
