# :busts_in_silhouette: Audience Architect -- Исследование целевой аудитории

Глубокое исследование ЦА через поведенческую экономику, JTBD, когнитивные триггеры и соматические маркеры.

## Команда

```
/aim audience https://example.com
```

## Выходной файл

`AUDIENCE-RESEARCH.md`

### Ключевые разделы

- **Макросегменты** -- 3-5 крупных сегментов с описанием
- **Микропортреты** -- детальные персоны с демографией, психографией, поведением
- **JTBD (Jobs to Be Done)** -- какие «работы» нанимает ваш продукт
- **Когнитивные триггеры** -- какие искажения и эвристики влияют на решение о покупке
- **Слова-магниты и стопперы** -- какие слова привлекают и отталкивают каждый сегмент
- **Ценообразование** -- готовность платить, ценовые якоря, модели

## Когда использовать

- Запуск нового продукта -- понять, кому продавать
- Низкая конверсия -- возможно, говорите не с той аудиторией
- Перед созданием рекламы и контента
- Пересмотр ценообразования

## Входные данные

| Параметр | Обязательный | Описание |
|----------|-------------|----------|
| URL сайта | Да | Адрес сайта для определения ЦА |

## Зависимости

Опционально читает: `MARKETING-AUDIT.md`, `COMPETITOR-REPORT.md`, `BRAND-VOICE.md`. Его выходной файл используется скиллами: emails, social, ads, headlines.

## Установка

Скилл устанавливается вместе со всем набором: `claude install-skills` из корня репозитория.
