# :envelope: Email Sequences -- Генерация email-цепочек

Создаёт готовые email-последовательности для всех этапов воронки: welcome, nurture, sales, retention.

## Команда

```
/aim emails https://example.com
```

## Выходной файл

`EMAIL-SEQUENCES.md`

### Ключевые разделы

- **Welcome-серия** -- 5-7 писем для новых подписчиков
- **Nurture-серия** -- прогрев и построение доверия
- **Sales-серия** -- продающая цепочка с дедлайнами и оффером
- **Retention / Win-back** -- возврат неактивных пользователей
- **Шаблоны писем** -- тема, прехедер, тело, CTA для каждого письма
- **Тайминг** -- когда отправлять каждое письмо

## Когда использовать

- Запуск email-маркетинга с нуля
- Текущие цепочки не конвертируют
- Новый продукт или оффер
- Нужно быстро подготовить серию писем

## Входные данные

| Параметр | Обязательный | Описание |
|----------|-------------|----------|
| URL сайта | Нет | Если не указан, использует данные из существующих файлов |

## Зависимости

Опционально читает: `BRAND-VOICE.md`, `AUDIENCE-RESEARCH.md`, `MARKETING-AUDIT.md`. Качество писем значительно выше при наличии этих файлов.

## Установка

Скилл устанавливается вместе со всем набором: `claude install-skills` из корня репозитория.
