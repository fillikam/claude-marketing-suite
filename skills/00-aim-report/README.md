# :bar_chart: Market Report -- Мета-оркестратор

Запускает все 9 скиллов последовательно и собирает результаты в единый сводный отчёт.

## Команда

```
/aim report https://example.com
```

## Выходной файл

`MARKETING-REPORT.md`

### Ключевые разделы

- **Executive Summary** -- краткие выводы для руководства
- **Сводный скоринг** -- оценки по всем направлениям в одной таблице
- **Топ-10 приоритетов** -- самые важные действия с оценкой impact
- **Стратегическая карта** -- связь между проблемами и решениями
- **План действий 30/60/90 дней** -- пошаговый roadmap

## Когда использовать

- Нужен полный маркетинговый анализ «под ключ»
- Готовите отчёт для клиента или руководства
- Хотите запустить все скиллы одной командой

## Входные данные

| Параметр | Обязательный | Описание |
|----------|-------------|----------|
| URL сайта | Да | Адрес анализируемого сайта |

## Зависимости

Этот скилл является оркестратором и запускает все остальные скиллы. Он читает выходные файлы всех 9 скиллов для формирования сводного отчёта.

## Установка

Скилл устанавливается вместе со всем набором: `claude install-skills` из корня репозитория.
