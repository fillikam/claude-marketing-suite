# ⚡ Быстрый старт

Пошаговая инструкция по установке и первому запуску Claude Marketing Suite.

---

## 🟢 Способ A: Самый простой (без кода, без терминала)

> Подходит для всех. Нужен только браузер и подписка Claude Pro.

### Шаг 1: Скачайте архив

На главной странице репозитория нажмите зелёную кнопку **Code** → **Download ZIP**.

Сохраните файл `claude-marketing-suite-master.zip` на компьютер.

### Шаг 2: Откройте Claude

Зайдите на [claude.ai](https://claude.ai) и начните новый чат.

### Шаг 3: Прикрепите ZIP и отправьте промпт

Нажмите иконку 📎 (скрепка) внизу чата, выберите скачанный ZIP-файл и напишите:

```
Установи все .skill файлы из этого архива как скиллы Claude.
Это маркетинговый пакет из 10 скиллов. Распакуй архив,
найди все файлы с расширением .skill и установи каждый.
После установки покажи список всех установленных скиллов
и объясни какую команду писать для каждого.
```

### Шаг 4: Проверьте

После установки напишите в том же чате:

```
/aim audit https://google.com
```

Если Claude начал анализ — всё работает! 🎉

## 🟡 Способ B: Установка скиллов по одному

> Если не нужен весь пакет — установите только нужные скиллы.

### Шаг 1: Скачайте нужный .skill файл

Перейдите в нужную папку в `skills/`, например:
- [`skills/01-aim-audit/`](skills/01-aim-audit/) → скачайте `aim-audit.skill`

Чтобы скачать: нажмите на файл → кнопка **Download** (⬇️) справа вверху.

### Шаг 2: Прикрепите к чату Claude

В [claude.ai](https://claude.ai):
1. Начните новый чат
2. Нажмите 📎 → выберите скачанный `.skill` файл
3. Напишите: `Установи этот скилл`
4. Готово!

### Какие скиллы ставить первыми?

| Приоритет | Файл | Зачем |
|-----------|------|-------|
| ⭐ Первый | `aim-audit.skill` | Базовый аудит — с него всё начинается |
| ⭐ Второй | `aim-landing.skill` | CRO-анализ лендингов |
| ⭐ Третий | `aim-report.skill` | Собирает всё в один отчёт |
| По желанию | остальные 7 скиллов | Ставьте по мере необходимости |

## 🔵 Способ C: Установка одним промптом в Claude Code

> Для тех, кто использует Claude Code (CLI в терминале).

Откройте Claude Code и вставьте:

```
Скачай репозиторий https://github.com/Fillikam/claude-marketing-suite
и установи из него все .skill файлы как скиллы.
```

Claude Code сам склонирует репо, найдёт все `.skill` файлы и установит их.

## 🟣 Способ D: Через терминал вручную

> Для разработчиков, которым нравится контроль.

```bash
# 1. Установите Claude Code (если ещё нет)
npm install -g @anthropic-ai/claude-code

# 2. Склонируйте репозиторий
git clone https://github.com/Fillikam/claude-marketing-suite.git
cd claude-marketing-suite

# 3. Запустите Claude Code
claude

# 4. Запустите первый аудит
/aim audit https://your-site.com
```

## Проверка установки

После любого способа установки напишите в чате:

```
/aim audit https://your-site.com
```

Claude проанализирует сайт и создаст файл `MARKETING-AUDIT.md` с:
- Скорингом по 12 категориям
- Списком проблем с приоритетами
- Рекомендациями с оценкой revenue impact
- Планом действий на 30/60/90 дней

## Рекомендуемый порядок запуска

Для максимального качества запускайте скиллы в таком порядке:

1. `/aim audit https://site.com` -- базовый аудит
2. `/aim landing https://site.com` -- CRO-аудит лендинга
3. `/aim competitors https://site.com` -- конкурентный анализ
4. `/aim brand https://site.com` -- голос бренда
5. `/aim audience https://site.com` -- исследование аудитории
6. `/aim emails` -- email-последовательности (использует данные из шагов 1-5)
7. `/aim social` -- контент-план (использует данные из шагов 1-5)
8. `/aim ads` -- рекламные кампании (использует данные из шагов 1-5)
9. `/aim headlines` -- заголовки (использует данные из шагов 1-5)
10. `/aim report` -- сводный отчёт (собирает все файлы)

Или запустите сразу всё одной командой:

```
/aim report https://your-site.com
```

## Часто возникающие проблемы

### Скилл не распознаётся

Убедитесь, что вы выполнили `claude install-skills` из корня репозитория.

### Файл не создаётся

Проверьте, что у Claude Code есть права на запись в текущую директорию.

### Недостаточно данных для анализа

Некоторые скиллы работают лучше, когда уже есть выходные файлы от других скиллов. Следуйте рекомендуемому порядку запуска.
